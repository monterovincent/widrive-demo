# WiDrive Fleet Dashboard — Demo

A production-quality fleet management dashboard demo built to showcase 
what WiDrive's product experience looks like with live data connected.

## The Problem This Solves
When a new user signs up to WiDrive, every screen shows "No data available." 
This demo shows exactly what the product looks like once trucks are connected 
via OBD-II — giving potential customers confidence before they commit.

## What's Inside
- **Dashboard** — Fleet health score, live critical alert banner, 
  vehicle status list and component health bars
- **Vehicles** — Filterable grid of all 8 vehicles with clickable 
  side panel showing individual health stats
- **Maintenance** — Full service booking flow with animated 4-step 
  confirmation sequence

## Tech Stack
- React 19 + Vite
- Tailwind CSS v3
- Mock data (no backend required)
